# Project Documentation

## System Design Diagrams

This directory contains the key architectural and design diagrams for the project.

### Available Diagrams

1. **[Domain Model](diagrams/domain-model.md)** - Conceptual model showing the key entities and relationships in the problem domain
2. **[Component Diagram](diagrams/component-diagram.md)** - High-level system architecture showing components and their interactions
3. **[ER Model](diagrams/er-model.md)** - Database schema design with entities, attributes, and relationships

### Diagram Formats

All diagrams are created using Mermaid syntax, which renders automatically on GitHub. They can also be:
- Exported to images using various Mermaid tools
- Edited using the [Mermaid Live Editor](https://mermaid.live/)
- Integrated into documentation generators

### How to View

Simply click on any of the diagram files above. GitHub will automatically render the Mermaid diagrams.

### How to Edit

1. Click on the diagram file you want to edit
2. Click the "Edit" button (pencil icon)
3. Modify the Mermaid code within the code blocks
4. Preview changes using the "Preview" tab
5. Commit your changes
